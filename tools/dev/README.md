# Вспомогательные скрипты разработки (Development & Demo Scripts)

> [!WARNING]
> Данные скрипты содержат демонстрационные/захардкоженные значения и предназначены только для тестирования или отладки в локальной среде разработки. **Не используйте их в рабочей (production) среде без предварительной проверки и редактирования.**

### Описание скриптов:
* `check_config.py` — проверяет тестовый конфиг для конкретного хоста (`Windows-Agent-1`).
* `check_pack_tags.py` — проверяет привязку демонстрационного пакета к тегам.
* `check_resultlogs.py` — выводит результаты логов.
* `check_tags.py` — проверяет тестовые метки (`метка 1`).
* `delete_pack.py` — удаляет конкретный демонстрационный пакет запросов.
* `delete_all_cmdb.py` — деструктивный скрипт очистки CMDB (пакетов/запросов).
